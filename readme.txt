AJ Review offers a variety of features for creating professional reviews.
You can manage and utilize your reviews more systematically.

== Description ==
== Review Group ==
* You can categorize reviews to manage them.

== Review ==
* You can link reviews to Post/Page to present review information where you want to go.
* Items about reviews can be set up and scored by the user himself.
* These scores are summed up as total points to define the score for the review target.

== Rank(Review List) ==
* Reviews can be expressed in a ranking table by score.
* Sort by specific evaluation items.
* Shortcodes are provided for a variety of options. [Shortcodes](https://2p1d.com/aj-%EB%A6%AC%EB%B7%B0/aj-%EB%A6%AC%EB%B7%B0-short-codes%E2%80%8B/)

== Widget ==
* We also offer Widget to present filtered reviews on Sidebar.

== Installation ==
1. Upload the **aj-review** folder to the directory **/wp-content/plugins/**.
2. Activate the plugin using the ‘**Plugins**’ menu in WordPress.

== Frequently Asked Questions ==
= Is the number of Review registrations limited? =
No, there is no limit on the number.

= Do you also offer Widget? =
Yes, also provides widget. You can filter the Sidebar to display the desired Review.
